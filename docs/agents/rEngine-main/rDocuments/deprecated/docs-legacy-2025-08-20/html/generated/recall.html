Recall.js - Instant Memory Lookup for Agents

Purpose & Overview

The recall.js script is a powerful tool designed to provide instant memory lookup capabilities for software agents. It enables agents to quickly search and retrieve relevant information from various sources, including extended context, agent memory, and task records. This script is a crucial component in building intelligent and self-aware agents that can efficiently access and utilize their accumulated knowledge.

Technical Architecture

The recall.js script is structured as follows:

  1. Entry Point: The script starts by checking the command-line arguments to determine if the script is being run directly (as a CLI tool) or imported as a module.
  2. Dependencies: The script imports necessary modules, such as fs/promises and path, to interact with the file system.
  3. Configuration: The script checks for a --silent flag in the command-line arguments to enable a "silent mode" for the Memory Controlled Process (MCP).
  4. Search Process: The fastRecall function is the core of the script, which performs the following steps:
    • Defines the priority order for searching: extended context, agent memory, and tasks.
    • Reads and parses the relevant JSON files from the file system.
    • Calculates the relevance of each search result based on the provided query.
    • Sorts the results by relevance and recency.
    • Displays the top 5 results, with additional information such as source, date, status, and content preview.
  5. Relevance Calculation: The calculateRelevance function is responsible for determining the relevance of a search result based on the provided query and text.

Dependencies

The recall.js script has the following external dependencies:

  • fs/promises: For asynchronous file system operations.
  • path: For working with file paths.
  • fileURLToPath: For converting a file URL to a file path.

Key Functions/Classes

fastRecall(query)

  • Purpose: Performs a fast recall search using the provided query.
  • Parameters:
    • query (string): The search term.
  • Return Value: None. The function displays the search results directly to the console.

calculateRelevance(query, text)

  • Purpose: Calculates the relevance score of a search result based on the provided query and text.
  • Parameters:
    • query (string): The search term.
    • text (string): The text to be compared against the query.
  • Return Value: A number between 0 and 1, representing the relevance score.

Usage Examples

To use the recall.js script, you can run it from the command line with a search query:

node recall.js "menu system"

This will perform a search for the term "menu system" and display the top 5 most relevant results.

You can also run the script in silent mode, which suppresses the console output:

node recall.js "console split" --silent

Configuration

The recall.js script does not have any additional configuration options or environment variables. The only configuration option is the --silent flag, which enables the MCP (Memory Controlled Process) mode.

Error Handling

The recall.js script handles errors gracefully by catching any exceptions that may occur during the search process. If an error occurs, it will be logged to the console with an appropriate error message.

Integration

The recall.js script is designed to be a standalone tool that can be used by software agents to quickly access and retrieve relevant information from their memory. It can be integrated into larger systems or agent-based architectures to provide instant memory lookup capabilities.

Development Notes

  • The script uses a priority-based search approach, first looking at the extended context (newest data), then the agent memory, and finally the tasks.
  • The calculateRelevance function uses a simple scoring mechanism based on word matching and exact matches to determine the relevance of each search result.
  • The script is designed to be efficient and scalable, as it only reads and processes the necessary files for each search query.
  • The script can be easily extended to support additional data sources or more advanced relevance calculation algorithms.

Overall, the recall.js script is a powerful and flexible tool that can significantly enhance the capabilities of software agents by providing them with instant access to their accumulated knowledge and experiences.
