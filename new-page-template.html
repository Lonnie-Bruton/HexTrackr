Shared Component Demo

This page demonstrates the HexTrackr shared component architecture:

  • ✅ Shared Header - Navigation loaded from scripts/shared/header.html
  • ✅ Shared Settings Modal - Modal loaded from scripts/shared/settings-modal.html
  • ✅ Import CSV Data link - Opens settings modal to Data Management tab
  • ✅ ServiceNow Integration - Available in Ticket Systems tab
  • ✅ Consistent UI - Same look and feel across all pages

Architecture Benefits:

  • One place to update navigation across all pages
  • Consistent settings modal across all pages
  • Easy to add new pages with shared components
  • No duplicate HTML or JavaScript code
  • Clear separation of shared vs page-specific code
