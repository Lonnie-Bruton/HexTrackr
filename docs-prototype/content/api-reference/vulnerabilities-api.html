Vulnerabilities API

The Vulnerabilities API provides endpoints for retrieving and importing vulnerability data.

Endpoints

GET /api/vulnerabilities

  • Description: Retrieves a list of all vulnerabilities.
  • Response: 200 OK
    [
  {
    "id": 1,
    "vulnerability_id": "CVE-2023-12345",
    "severity": "High",
    "status": "Open",
    "first_seen": "2023-10-01T12:00:00Z",
    "last_seen": "2023-10-27T14:00:00Z"
  }
]

POST /api/vulnerabilities/import

  • Description: Imports vulnerabilities from a CSV file.
  • Request Body: multipart/form-data with a single file field named csvfile.
  • Response: 200 OK
    {
  "message": "Import successful",
  "new_vulnerabilities": 10,
  "updated_vulnerabilities": 5
}
