JavaScript Reference

Detailed documentation of key JavaScript functions in the HexTrackr codebase.

Core Application Functions

debounce(func, wait, immediate)

File: scripts/ag-grid-responsive-config.js

Purpose: Creates a debounced function that delays invoking func until after wait milliseconds have elapsed since the last time the debounced function was invoked.

Parameters:

  • func - The function to debounce
  • wait - The number of milliseconds to delay
  • immediate - If true, trigger the function on the leading edge

Returns: The debounced function

Usage Example:
const debouncedSearch = debounce(performSearch, 300);

createVulnerabilityGridOptions()

File: scripts/ag-grid-responsive-config.js

Purpose: Creates and configures AG Grid options for vulnerability data display with responsive column definitions.

Parameters: None

Returns: AG Grid configuration object with column definitions and responsive settings

Usage Example:
const gridOptions = createVulnerabilityGridOptions();

validateAndPrepareDeviceData(deviceData)

File: scripts/validation-utils.js

Purpose: Validates device data structure and prepares it for database insertion or update operations.

Parameters:

  • deviceData - Object containing device information to validate

Returns: Object with validation results and cleaned data

Usage Example:
const result = validateAndPrepareDeviceData(formData);

Ticket Management Functions

refreshPageData()

File: scripts/pages/tickets.js

Purpose: Refreshes all ticket data on the page including table contents and summary statistics.

Parameters: None

Returns: Promise that resolves when data refresh is complete

Usage Example:
await refreshPageData();

showToast(message, type)

File: scripts/pages/tickets.js

Purpose: Displays a toast notification to the user with specified message and styling.

Parameters:

  • message - Text message to display
  • type - Toast type ('success', 'error', 'info', 'warning')

Returns: void

Usage Example:
showToast('Ticket saved successfully', 'success');

Shared Component Functions

initializeSettingsModal()

File: scripts/shared/settings-modal.js

Purpose: Initializes the global settings modal with all necessary event listeners and state management.

Parameters: None

Returns: void

Usage Example:
initializeSettingsModal();

saveSettingsData(settingsData)

File: scripts/shared/settings-modal.js

Purpose: Saves user settings to localStorage and applies them to the current session.

Parameters:

  • settingsData - Object containing user settings to save

Returns: Promise that resolves when settings are saved

Usage Example:
await saveSettingsData({theme: 'dark', notifications: true});
