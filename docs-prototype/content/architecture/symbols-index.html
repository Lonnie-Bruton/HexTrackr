HexTrackr Code Symbols Index

This document provides an index of key functions, variables, and components used throughout the HexTrackr application.

1. Backend (server.js)

Functions

  • initDb(): Initializes the SQLite database and ensures schema is up-to-date.

2. Frontend (scripts/)

pages/tickets.js

  • ticketManager: Main object for managing ticket operations.
  • loadTickets(): Fetches and displays tickets.
  • saveTicket(): Creates or updates a ticket.
  • deleteTicket(): Deletes a ticket.
  • viewTicket(): Displays ticket details in a modal.
  • exportData(): Exports ticket data to various formats.

pages/vulnerabilities.js

  • vulnManager: Main object for managing vulnerability operations.
  • loadVulnerabilityData(): Fetches and displays vulnerability data.
  • setupEventListeners(): Initializes all event listeners for the page.
  • initializeGrids(): Sets up the AG Grid instances.
  • renderDeviceCards(): Creates and displays device summary cards.
  • renderVulnerabilityCards(): Creates and displays vulnerability detail cards.
  • showDeviceModal(): Displays the device details modal.
  • showVulnerabilityModal(): Displays the vulnerability details modal.

shared/settings-modal.js

  • loadSettingsModal(): Injects the settings modal HTML into the page.
  • setupSettingsEventListeners(): Attaches event listeners to the modal's controls.
  • handleImport(): Manages the file import process.
  • handleBackup(): Manages the data backup process.
  • handleRestore(): Manages the data restore process.
  • handleClearData(): Manages the data clearing process.

3. HTML Structure (*.html)

Key IDs

  • ticketsTableBody: <tbody> for the tickets table.
  • vulnGrid: The main AG Grid container for vulnerabilities.
  • deviceCards: Container for the device summary cards.
  • vulnerabilityCards: Container for the vulnerability detail cards.
  • ticketModal: Modal for adding/editing tickets.
  • viewTicketModal: Modal for viewing ticket details.
  • deviceModal: Modal for viewing device details.
  • vulnerabilityModal: Modal for viewing vulnerability details.
  • settingsModalContainer: Container for the shared settings modal.
