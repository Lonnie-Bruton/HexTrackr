Frontend Architecture

The HexTrackr frontend is built with a mix of technologies to support its dual-purpose nature.

Frameworks

  • Ticket Management (tickets.html): Built with Bootstrap 5 for a clean, standard interface.
  • Vulnerability Management (vulnerabilities.html): Built with Tabler.io for its rich set of data visualization components and a modern dashboard aesthetic.

JavaScript Structure

  • scripts/pages/: Contains page-specific JavaScript files.
    • tickets.js: Logic for the ticketing system.
    • vulnerabilities.js: Logic for the vulnerability dashboard.
  • scripts/shared/: Contains JavaScript modules shared across pages.
    • settings-modal.js: A unified settings modal used on all pages.
  • scripts/utils/: Contains utility functions for common tasks like API calls and data formatting.
