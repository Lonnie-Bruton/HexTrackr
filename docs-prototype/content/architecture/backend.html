Backend Architecture

The backend is a Node.js application using the Express framework.

Key Components

  • server.js: The main entry point for the application. It sets up the Express server, defines middleware, and mounts the API routes.
  • API Routes: The REST API is defined to handle CRUD (Create, Read, Update, Delete) operations for tickets and vulnerabilities.
  • Middleware:
    • express.json(): For parsing JSON request bodies.
    • express.urlencoded(): For parsing URL-encoded request bodies.
    • Static file serving for the frontend assets.

File Uploads

  • The uploads/ directory is used for temporary storage of imported CSV files.
  • The backend processes these files and inserts the data into the database.
