Prerequisites

Before installing HexTrackr, ensure your system meets the following requirements:

System Requirements

  • Operating System: macOS, Linux, or Windows with WSL2
  • Docker: Version 20.0+ with Docker Compose
  • Memory: Minimum 2GB RAM available for containers
  • Storage: 1GB free disk space for application and data
  • Network: Internet access for initial image downloads

Required Software

  • Docker Desktop (recommended) or Docker Engine + Docker Compose
  • Git for repository management
  • Text Editor (VS Code, nano, vim) for configuration

Functional Testing

  1. Web Interface Access

    • Navigate to http://localhost:8080
    • Verify homepage loads correctly
    • Check navigation between Tickets and Vulnerabilities

  2. Database Operations

    • Create a test ticket
    • Upload a test CSV file (vulnerabilities)
    • Verify data persistence after container restart

  3. File Operations

    • Test PDF export functionality
    • Verify file upload restrictions
    • Check data directory permissions
