User Guide: Ticket Management

The Ticket Management page (tickets.html) is the central hub for creating, tracking, and managing remediation tasks within HexTrackr.

Interface Overview

The interface is designed for clarity and efficiency, providing all necessary tools on a single screen.

  1. Header: Contains the main navigation, a link to the Vulnerabilities page, and the settings icon.
  2. Action Buttons:
    • Add New Ticket: Opens a modal to create a new ticket.
    • Attach Documentation: Allows for attaching relevant documents to a ticket.
    • Export Buttons: A group of buttons to export the ticket data in various formats (CSV, Excel, JSON, PDF, HTML).
  3. Statistics Cards: Display a high-level overview of ticket status: Total, Open, Overdue, and Completed.
  4. Search and Filters:
    • A search bar allows for free-text searching across all ticket fields.
    • Dropdown filters for Status and Location allow for quick filtering of the ticket list.
  5. Tickets Table: The main component, displaying a list of all tickets. It is sortable by clicking on the column headers.
  6. Pagination: Controls at the bottom of the table allow for navigating through pages of tickets and changing the number of rows displayed per page.

Creating and Editing Tickets

Adding a New Ticket

  1. Click the "Add New Ticket" button.
  2. The "Add New Ticket" modal will appear.
  3. Fill in the required fields:
    • Dates: Date Submitted and Date Due.
    • Ticket Numbers: Hexagon Ticket # and Service Now # (if applicable).
    • Details: Site, Location, Supervisor, and Tech.
    • Status: Select the current status from the dropdown (e.g., Open, In Progress).
    • Devices: Add one or more device names. You can add, remove, and reorder devices to specify a boot sequence.
    • Notes: Add any additional details in Markdown format.
  4. Click "Save Ticket". The new ticket will be assigned a unique XT# and will appear in the table.

Editing an Existing Ticket

  1. In the tickets table, find the ticket you wish to edit.
  2. Click the pencil icon (<i class="fas fa-edit"></i>) in the "Actions" column for that row.
  3. The same modal used for adding tickets will appear, pre-filled with the existing ticket's data.
  4. Make your changes and click "Save Ticket".

Viewing Ticket Details

  1. Click the eye icon (<i class="fas fa-eye"></i>) in the "Actions" column of any ticket.
  2. The "View Ticket" modal will appear, displaying a formatted summary of the ticket in a terminal-style view. This content is generated from the ticket's data and is presented in Markdown.
  3. From this view, you can:
    • Copy Markdown: Copies the full Markdown content to your clipboard.
    • Download Bundle: Downloads a .zip file containing the ticket details in a Markdown file along with any attached documentation.
    • Edit Ticket: Closes the view modal and opens the edit modal for that ticket.

Deleting a Ticket

  1. Click the trash can icon (<i class="fas fa-trash"></i>) in the "Actions" column for the ticket you want to delete.
  2. A confirmation prompt will appear to prevent accidental deletion.
  3. Confirm the action to permanently remove the ticket from the database.

Sorting and Filtering

  • Sorting: Click on any column header (e.g., XT#, Date Due, Status) to sort the table by that column. Clicking again will reverse the sort order.
  • Filtering: Use the Status and Location dropdowns to narrow down the list of tickets. The table will update in real-time as you select filter options.
  • Searching: Type a keyword into the search bar to filter the table to only show rows containing that keyword.
