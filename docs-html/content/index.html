Welcome to HexTrackr Documentation

This portal contains all the documentation for the HexTrackr application, a dual-purpose cybersecurity management system for tracking tickets and vulnerabilities.

Whether you are a user, developer, or administrator, these documents provide the information you need to use, develop, and maintain HexTrackr effectively.

Documentation Sections

  • Getting Started: If you are new to HexTrackr, start here. This section provides instructions for setting up the application using Docker or for local development.

  • User Guides: Detailed guides on how to use the features of the application, including ticket management, vulnerability analysis, and data import/export.

  • API Reference: A complete reference for the HexTrackr REST API, including all endpoints, request/response formats, and usage examples.

  • Architecture: A deep dive into the technical architecture of the system, including the backend, frontend, database, and the vulnerability rollover mechanism.

  • Development: Information for developers who want to contribute to the project, including coding standards and contribution guidelines.

  • Project Management: An overview of the project's roadmap, sprints, and quality standards.

  • Security: Documentation related to the security of the HexTrackr application, including the vulnerability disclosure policy.
